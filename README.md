# Form Parse Activity

- Node.js
- Express.js
- mustache-express
- body-parser

### Build Steps:
- `mkdir <project-name>`
- `cd <project-name>`
- `gitignore node`
- `npm init --yes`
- `npm install express --save`
- `npm install mustache mustache-express --save`
- `npm install body-parser --save`
- `touch app.js`
- `touch README.md`
- `mkdir views`
- `cd views`
- `touch signup.mustache`
